Коротка інформація

GNU/Linux — це вільна та відкрита операційна система, що поєднує ядро Linux та програмне забезпечення проекту GNU. Використовується на серверах, комп’ютерах, мобільних пристроях та вбудованих системах.

Приклади команд Linux CLI

pwd — показує поточну директорію
echo — виводить текст у термінал
ls — показує вміст директорії
cd — змінює директорію
touch — створює новий файл
mkdir — створює нову папку
cp — копіює файли або папки
mv — переміщує або перейменовує файли
rm — видаляє файли
history — показує історію команд

BASH-скрипт авторизації

  #!/bin/bash
  echo "Введіть ім'я користувача:"
  read username
  echo "Введіть пароль:"
  read -s password
  if [ "$username" = "admin" ] && [ "$password" = "1234" ]; then
      echo "Авторизація успішна!"
  else
      echo "Невірне ім’я користувача або пароль"
  fi
